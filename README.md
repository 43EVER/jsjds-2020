#### 功能

1. 图形化编程（类似UE4的蓝图）
2. 用户可上传自定义模块



#### 学习内容

1. LLVM 的新手教程：粗略的了解一下，一门语言的实现流程
2. 编译原理的入门课程：知道编译器中的一些专业词汇和概念
   1. 跳过语法分析的算法部分
   2. 跳过语法指导翻译
   3. 跳过中间代码及代码优化部分
3. VUE 的学习，更优雅的实现
